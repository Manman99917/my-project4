# 打卡鸭 - 习惯日历小程序

一款可爱的习惯打卡记录小程序，帮助你养成良好的生活习惯。

## 功能特性

### 📝 事项管理
- **自定义事项**：支持添加任意事项（排便、洗头、喝水、运动等）
- **个性化设置**：
  - 自定义事项名称和图标（10+ emoji 可选）
  - 14 种精美卡通背景样式（渐变色 + 装饰图案）
  - 自定义提醒间隔时间（小时）
- **默认事项**：内置排便和洗头两个事项，不可删除
- **长按删除**：自定义事项支持长按删除

### ⏰ 打卡记录
- **一键打卡**：点击事项卡片弹出打卡按钮，确认即可记录
- **防误触**：短时间内重复打卡会提示
- **有趣反馈**：打卡成功随机显示鼓励文案
- **震动反馈**：打卡成功时手机短震动提示

### 📅 日历视图
- **月视图展示**：清晰查看每月打卡情况
- **图标标记**：每个日期显示最多 2 个事项图标，超过显示 `+` 角标
- **点击查看详情**：点击日期查看当天所有打卡记录
- **滑动切换月份**：左右滑动快速切换月份
- **记录管理**：支持删除单条历史记录

### 🎨 视觉设计
- **卡通可爱风格**：粉嫩配色 + 波点/波浪/条纹装饰图案
- **14 种背景样式**：
  - 渐变色：粉橙、蓝紫、绿薄荷、黄橙、桃粉、紫蓝、薄荷绿、奶橙、天蓝、樱花粉
  - 纯色：奶黄、薄荷、淡紫、粉嫩
- **吉祥物"嘟嘟"**：
  - 首页底部随机显示图片和可爱文案
  - 每次切换 tab 自动更换
  - 点击吉祥物区域立即更换

### 📊 数据统计
- **最近记录**：显示上次打卡时间和距今间隔
- **提醒功能**：超过设定时间间隔自动显示提醒文案

## 技术栈

- **框架**：微信小程序原生开发
- **存储**：`wx.storage` 本地存储
- **架构**：
  - `utils/RecordService.js`：数据服务层，封装所有数据操作
  - `pages/`：页面层，负责 UI 交互
  - `config/`：常量配置

## 数据存储

使用微信小程序本地存储 `wx.storage`：

- `event_list`：事项列表配置
- `latest_{eventId}`：事项的最近一次打卡时间
- `history_{eventId}`：事项的完整打卡历史（时间戳数组）

## 开发规范

### 代码风格
- 使用 JSDoc 格式注释
- 所有存储操作使用 try-catch 包裹
- 禁止在 Page 层直接操作存储，统一通过 RecordService
- 常量统一提取，避免魔法字符串

### 错误处理
- 所有数据操作添加错误处理
- 失败时通过 `wx.showToast` 友好提示用户
- Console 记录详细错误信息便于调试

## 使用说明

### 添加事项
1. 点击"+ 添加事项"按钮
2. 输入名称、选择图标和背景样式
3. 设置提醒间隔（小时）
4. 点击"确定"保存

### 打卡
1. 点击事项卡片
2. 弹出打卡确认窗
3. 点击圆形"打卡"按钮完成记录

### 编辑事项
1. 点击事项卡片弹出打卡窗
2. 点击底部"设置提醒间隔"
3. 修改名称、图标、背景样式或提醒间隔
4. 点击"保存"

### 删除事项
- 长按自定义事项卡片
- 确认删除（会同时删除所有历史记录）
- 默认事项不可删除

### 查看历史
1. 切换到"日历" tab
2. 查看月视图，有图标的日期表示有打卡记录
3. 点击日期查看详情
4. 在详情中可删除单条记录
5. 左右滑动切换月份
